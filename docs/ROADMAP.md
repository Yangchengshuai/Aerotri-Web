## 短期规划 (0-1 个月)

### 核心功能增强

- [ ] **大场景分 Tile 重建支持**
  - 支持超大场景的分块处理
  - 自动 Tile 划分算法
  - LOD (Level of Detail) 支持

- [ ] **ROI (感兴趣区域) 重建**
  - 用户可选择特定区域进行重建
  - 矩形/多边形 ROI 选择

- [ ] **大场景 3DGS 分 chunk 训练**
  - 大场景自动分块训练
  - Chunk 边界无缝拼接
  - 分布式训练支持
  - Chunk 合并和优化

### 性能优化

- [ ] **算法性能优化**
  - CUDA 加速优化
  - 内存占用优化
  - 多 GPU 负载均衡
  - 缓存机制改进

- [ ] **前端性能优化**
  - 大数据列表虚拟滚动
  - 3D 场景 LOD 加载
  - 组件懒加载
  - 资源预加载策略

## 中期规划 (1-2 个月)

### 新工作流支持

- [ ] **手持激光雷达工作流**
  - 支持激光雷达点云输入
  - 点云与图像融合
  - 激光雷达标定工具
  - 点云优化和滤波

- [ ] **大场景分 Tile 重建**
  - 自动 Tile 分割
  - 并行处理
  - Tile 间边界优化
  - 无缝拼接

- [ ] **LOD 加载**
  - 多级细节层次
  - 视点相关加载
  - 流式传输
  - 缓存管理

### 用户界面增强

- [ ] **可视化改进**
  - 更丰富的 3D 可视化选项
  - 实时预览功能
  - 交互式参数调整
  - 对比工具增强

- [ ] **协作功能**
  - 多用户支持
  - 权限管理
  - 项目共享
  - 审批流程
